# Icons Needed

Create PNG icons at these sizes:

| File | Size | Required For |
|------|------|--------------|
| `icon16.png` | 16x16 | Browser favicon |
| `icon32.png` | 32x32 | Toolbar (Windows) |
| `icon48.png` | 48x48 | Extensions page |
| `icon128.png` | 128x128 | Chrome Web Store |

## Design Suggestions

- Use the Golden Codex scroll/book motif
- Colors: Purple (#7b2d8e) to Gold (#c9a227) gradient
- Simple, recognizable at small sizes
- Transparent background

## Quick Generation

Use an online tool like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

Or create in Figma/Illustrator and export at each size.

## Temporary Testing

The extension will work without icons (using default browser icon).
Add real icons before publishing to stores.
